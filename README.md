# 🧠 Burnout AI Detector (Promptathon 2026)

An AI-powered diagnostic tool designed to detect early signs of occupational burnout using behavioral data and Natural Language Processing (NLP). This project focuses on **Explainable AI (XAI)** to provide users with actionable insights into their mental well-being.

---

## 🚀 Key Features
* **Behavioral Analysis:** Uses NLP to identify patterns of exhaustion, cynicism, and reduced professional efficacy.
* **XAI Integration:** Doesn't just give a score; it explains *why* the AI flagged specific inputs.
* **Prompt Engineering:** Utilizes Chain-of-Thought (CoT) prompting to minimize false positives in stress detection.
* **Modern UI:** A responsive, clean dashboard built for high-stress users.

## 📂 Project Structure
* `frontend/`: The user interface (HTML, Tailwind CSS, JavaScript).
* `backend/`: The logic layer (Server and API integration).
* `prompt_journey.md`: Documentation of the prompt iterations used for the model.
