# Prompt Engineering Journey

### Phase 1: Idea Refinement
**Prompt:** "Suggest a high-impact AI project for burnout detection that focuses on task fragmentation and workload intensity."

### Phase 2: Logic Generation
**Prompt:** "Write a JavaScript function that calculates burnout risk based on four factors: Workload, Task Count (Fragmentation), Focus Hours, and Breaks. Return a score out of 100 and specific reasons."

### Phase 3: Full Stack Implementation
**Prompt:** "Create a Node.js Express backend and a modern HTML/CSS frontend that communicates with it to display the burnout results."
